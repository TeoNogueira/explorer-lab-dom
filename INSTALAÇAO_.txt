 O que são módulos ecmascript?

 R= É quando queremos importar ou exportar arquivos para o javascript (.js), no exemplo desse lab-explorer temos o arquivo CSS que está sendo usado dentro do main.js. 


 para usar módulos no Javascript, defina type dentro da tag script no html,  exemplo:

 <script type="module" src="./src/main.js"></script>

Para que serve o npm Run Dev?

 O npm é uma ferramenta de linha de comando que ajuda a interagir com plataformas online, como navegadores e servidores. Essa utilidade auxilia na instalação e desinstalação de pacotes, gerenciamento da versões e gerenciamento de dependências necessárias para executar um projeto.
